    Resizes an image to the specified size and adds an extra transparent margin at all sides of the image.

    Declaration

    Swift

    func imageWithSize(size: CGSize, extraMargin: CGFloat) -> UIImage?

    Parameters

    size

    the size we desire to resize the image to.
    extraMargin

    the extra transparent margin to add to all sides of the image.

    Return Value

    the resized image. The extra margin is added to the input image size. So that the final image’s size will be equal to: CGSize(width: size.width + extraMargin * 2, height: size.height + extraMargin * 2)
